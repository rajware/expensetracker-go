# Manifest to deploy Expense Tracker with a PostgreSQL database
#
# Edit the Secret called tracker-postgres-secret:
#   hmacKey: a string used to sign the auth cookie.
#   dbPassword: The password for a user called trackeruser on PostGres
#   dbUrl: A postgres URL in the form:
#              postgres://trackeruser:PASSWORD@tracker-postgres-db-svc:5432/expensetrackerdb
#
# If your cluster has an Ingress controller, please see below:
#
# By default, the Service called tracker-postgres-fe-svc has `type: NodePort`.
# Change this to `type: ClusterIP` if the cluster has an Ingress controller.
# Also, remove the spec.ports[].nodePort property.
#
# Finally, uncomment the Ingress called tracker-postgres-ingress.
# Edit the Ingress called tracker-postgres-ingress:
#    host: The hostname used to access Expense Tracker
